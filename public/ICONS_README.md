# تعليمات إضافة الأيقونات

يجب إضافة الأيقونات التالية إلى مجلد `public`:

1. **icon-192.png** - أيقونة بحجم 192x192 بكسل
2. **icon-512.png** - أيقونة بحجم 512x512 بكسل
3. **badge-72.png** - شارة بحجم 72x72 بكسل

يمكنك إنشاؤها باستخدام:
- أي برنامج تصميم (Photoshop, Figma, Canva)
- أو موقع https://www.favicon-generator.org/
- أو استخدام أيقونة بسيطة مؤقتاً

الأيقونة يجب أن تحتوي على:
- خلفية زرقاء (#1e40af)
- حرف "ذ" أو "IT" باللون الأبيض
- مربعة الشكل مع زوايا دائرية
